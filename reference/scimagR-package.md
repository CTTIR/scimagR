# scimagR: Spinal Cord Injury Imaging Pipeline for MRI and CT Analysis in R

An end-to-end pipeline for longitudinal MRI and CT analysis in spinal
cord injury (SCI) research. Provides R wrappers for the Spinal Cord
Toolbox (SCT) including SCIsegV2 lesion segmentation, DICOM
anonymization and NIfTI conversion tools, SCI-specific clinical metrics
(MSCC, compression ratio, tissue bridge quantification), implant
artifact grading, and pipeline orchestration with built-in quality
control. Integrates with workflowr for reproducible, publication-ready
neuroimaging analyses.
