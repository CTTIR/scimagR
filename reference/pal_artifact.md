# Artifact Grade Colour Palette

Named character vector of 5 colours for MRI artifact grades: none
(green), mild (yellow), moderate (orange), severe (red), non-evaluable
(grey).

## Usage

``` r
pal_artifact
```

## Format

An object of class `character` of length 5.

## Examples

``` r
pal_artifact
#>          none          mild      moderate        severe non-evaluable 
#>     "#4CAF50"     "#FFC107"     "#FF9800"     "#F44336"     "#9E9E9E" 
```
