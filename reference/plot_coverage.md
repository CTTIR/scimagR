# Plot Modality Coverage Matrix

Creates a tile plot showing modality coverage per patient and session,
with optional artifact grade overlay.

## Usage

``` r
plot_coverage(registry)
```

## Arguments

- registry:

  Data frame. Validated registry.

## Value

A ggplot2 object.

## Examples

``` r
if (FALSE) { # \dontrun{
plot_coverage(registry)
} # }
```
